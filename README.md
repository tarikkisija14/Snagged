> A modern thrift shop web application for discovering and selling pre-loved items

## 📋 Overview

Snagged is a full-stack thrift shop platform that connects buyers and sellers of second-hand goods. Built with modern web technologies, it provides a seamless shopping experience for sustainable fashion and lifestyle enthusiasts.

## ✨ Features

- 🔍 Browse and search thrift items
- 🛒 Shopping cart functionality
- 👤 User authentication and profiles
- 📦 Product management
- 💳 Secure checkout process
- 📱 Responsive design for all devices

## 🛠️ Tech Stack

### Backend
- **ASP.NET Core** - RESTful API
- **C#** - Primary programming language
- **SQL Server** - Database management
- **Entity Framework Core** - ORM

### Frontend
- **Angular** - Modern web framework
- **TypeScript** - Type-safe development
- **RxJS** - Reactive programming
- **Angular Material** - UI components

### Development Tools
- **Visual Studio** - Backend IDE
- **Webstorm** - Frontend development
- **SQL Server Management Studio** - Database administration
- **Angular CLI** - Project scaffoling and tooling
